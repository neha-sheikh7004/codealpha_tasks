# Global YouTube Statistics: Data Analysis, Visualization, and Insights

## Project Overview

This project analyzes the Global YouTube Statistics dataset to uncover trends, patterns, and insights about YouTube channels worldwide. The analysis includes data cleaning, exploratory data analysis (EDA), and multiple visualizations to better understand channel performance, audience reach, and content trends.

## Objectives

* Clean and prepare the dataset for analysis.
* Explore channel statistics across different countries and categories.
* Create meaningful visualizations to identify trends and patterns.
* Generate insights that help understand factors influencing channel growth and popularity.

## Dataset

The dataset contains information about top YouTube channels, including:

* Subscribers
* Video Views
* Uploads
* Channel Categories
* Channel Types
* Country Information
* Channel Creation Details
* Earnings and Ranking Metrics

## Tools and Libraries

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Cleaning

The following preprocessing steps were performed:

* Handled missing values.
* Checked and removed duplicate records.
* Reviewed data types for consistency.
* Prepared data for visualization and analysis.

## Visualizations Created

1. Top 10 Countries by Number of Channels
2. Top 10 Categories by Number of Channels
3. Top Channel Types on YouTube
4. Top 10 Channels by Subscribers
5. Top Countries by Total Subscribers
6. Channel Creation Trend by Year
7. Subscribers vs Video Views
8. Uploads vs Subscribers

## Key Insights

* The United States and India have the highest number of top YouTube channels.
* Entertainment and Music are among the most dominant content categories.
* A small number of channels hold exceptionally large subscriber bases.
* Subscriber count shows a positive relationship with video views.
* Upload frequency alone does not guarantee subscriber growth.
* The number of successful YouTube channels has increased significantly over time.

## Conclusion

This analysis highlights major trends within the YouTube ecosystem and demonstrates how data visualization can be used to uncover meaningful insights. The project provides a better understanding of channel growth, audience engagement, and content distribution across different regions and categories.
