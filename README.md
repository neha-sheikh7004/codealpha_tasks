# Flipkart Mobile Data Analysis and Web Scraping Project

## Project Overview

This project focuses on web scraping, data cleaning, data analysis, and visualization using Python. Mobile phone data was scraped from Flipkart and analyzed to understand pricing, ratings, and brand distribution.

## Technologies Used

* Python
* BeautifulSoup
* Pandas
* Matplotlib
* Requests

## Project Workflow

1. Scraped mobile phone data from Flipkart using BeautifulSoup.
2. Extracted:

   * Product Names
   * Prices
   * Descriptions
   * Reviews/Ratings
3. Cleaned the dataset by:

   * Removing duplicate values
   * Handling missing values
   * Removing symbols from price column
   * Converting datatypes for analysis
4. Saved the cleaned dataset into CSV format.
5. Performed data visualization and analysis using Matplotlib.

## Visualizations Included

* Top 10 Highest Priced Phones
* Top Rated Phones
* Brand Distribution Pie Chart

## Project Outcome

This project helped in understanding the complete data analysis workflow from web scraping to data visualization using Python.
