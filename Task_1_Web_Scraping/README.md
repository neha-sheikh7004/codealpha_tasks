# Flipkart Mobile Data Analysis and Web Scraping Project

Web scraping and data analysis project using Python, BeautifulSoup, Pandas, and Matplotlib on Flipkart mobile phone data.

## Project Overview

This project focuses on collecting, cleaning, analyzing, and visualizing mobile phone data from Flipkart. Data was scraped from multiple pages, processed using Pandas, and analyzed to generate insights related to mobile prices, ratings, and brand distribution.

## Technologies Used

- Python
- BeautifulSoup
- Requests
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Workflow

### 1. Web Scraping
- Scraped mobile phone data from multiple Flipkart pages.
- Extracted:
  - Product Name
  - Price
  - Description
  - Rating

### 2. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Removed currency symbols and commas from prices.
- Converted data into appropriate data types for analysis.

### 3. Data Analysis
- Sorted products based on prices and ratings.
- Compared products using different analytical techniques.
- Explored trends and patterns within the dataset.

### 4. Data Visualization
Created visualizations using Matplotlib, including:
- Top 10 Highest-Priced Mobile Phones
- Brand Distribution Analysis
- Price and Rating Insights

### 5. Data Export
- Exported the cleaned dataset into CSV format for further analysis and reporting.

This project provided practical experience in:
